# CLAUDE.md — Madrona Studio Website

## Project Identity

**Site:** Madrona Studio (singular, not "Studios")
**First Property:** The Ferrari Covenant by E.M. Voss
**Author:** ET (sherpaq@gmail.com)
**Creative Voice:** Elena Ferrante's interiority and emotional style — all writing and outlining should follow this register

## Dual Audience

The site serves two audiences simultaneously:
- **Readers:** atmospheric immersion, cinematic pacing, emotional entry into the novel
- **Film/TV Producers:** recognizing a cinematic property with scope, production value, and adaptation potential

## Framework & Stack

- **Astro** v6.0.8 — static site generator, zero JS by default, ships pre-built HTML
- **Tailwind CSS** v3.4.19 — utility-first CSS via design tokens in `tailwind.config.mjs`
- **No React** — not currently needed. Can add later with `npx astro add react` if interactive islands become necessary
- **Fonts:** Cormorant Garamond (display/serif) + Inter (body/sans-serif) via Google Fonts
- **Assets:** Hosted on ImageKit.io (`ik.imagekit.io/a7tech/madronastudio/`)
- **Deployment:** AWS Amplify (config: `amplify.yml`)

## Key Commands

## The Novel's 8-Section Plot Arc

All content planning maps to these beats:

1. **The Smuggler's Return** — April 1940, SS Rex/Rome
2. **The Last Race of the Old World** — April 1940, Brescia
3. **The Command Performance** — May 1941, Modena
4. **The Night Shifts** — Sept-Dec 1943, Maranello/Modena
5. **The Firestorm Escape** — Feb 14, 1944, Modena
6. **The Milan Transformation** — Spring 1944, Milan
7. **The Great Seizure** — April 1945, Maranello
8. **The Ghost's Last Race** — April 1945, Via Emilia/Taro River

**Home page carries beats 1, 4, 5, 8** (the arc's cardinal points).
**The World page carries all 8** as geographic progression.

## Design Principles

- The site is cinematic. Scrolling is pacing. Images are shots. Silence between sections is the held breath between scenes.
- Nothing that calls attention to itself as animation — transitions are subtle (800ms reveal, cubic-bezier easing)
- Text never stretches to full width — max-width: 42rem for prose, 56rem for wide passages
- The `.voice-elena` class sets Cormorant Garamond italic for interior/Ferrante-register text
- Mobile uses 100dvh for true viewport height on hero sections

## ImageKit Configuration Note

ImageKit has a referrer allowlist that blocks embedded requests from `file://` origins. For local development, either:
- Serve via `localhost` (use `npm run dev` or `python3 -m http.server`)
- Update ImageKit settings to allow `localhost` and your production domain
